# Support

NextDog is a free, open-source project. The best way to get help is through GitHub Issues.

- **Found a bug?** Open a [bug report](https://github.com/AlberichLabs/nextdog/issues/new/choose).
- **Have an idea or feedback?** Open a [feedback / feature request](https://github.com/AlberichLabs/nextdog/issues/new/choose).

Before filing, a quick search of [existing issues](https://github.com/AlberichLabs/nextdog/issues) may turn up an answer or a related report you can add to.

For setup and usage, see the [README](./README.md).
